Frontend Interview Questions

Q1: What is HTML?

HTML stands for HyperText Markup Language. It is the standard markup language for creating web pages and web applications.

Q2: What is CSS?

CSS stands for Cascading Style Sheets. It is used for describing the presentation of a document written in HTML or XML, including the colors, fonts, and spacing.

Q3: What is JavaScript?

JavaScript is a high-level, interpreted programming language that is widely used to make web pages interactive. It can be executed in the browser without the need for compilation.
